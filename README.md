# 酷安体验优化脚本

## 📌 项目简介
本脚本旨在为酷安用户提供更**简洁的应用体验**，通过管理非核心组件与清理冗余资源实现优化。  
**注意**：仅供学习交流，禁止商用或二次分发。

---

## ⚙️ 使用前提
- 需 **Root权限** 执行
- 仅适用于 `com.coolapk.market` 官方版本(15.2.2)

---

## 🛠️ 使用方法

### 步骤 1：下载脚本
将 [`FUCK_COOL_AD.sh`](FUCK_COOL_AD.sh) 保存至设备 `/sdcard/` 目录。

### 步骤 2：执行脚本
- **Root用户**：使用MT管理器(SU)执行此脚本
